Тег <area>

Для того, чтоб лучше понимать , как работает тег <area> давайте разберем, что же такое КАРТА ИЗОБРАЖЕНИЙ (Image Map)?

Карта изображений (Image Map)

Карта изображений (Image Map) позволяет привязывать ссылки к фрагментам изображения. Щелкая мышью на отдельных частях изображения, пользователь может переходить по той или иной ссылке на разные Web-страницы.

  • Для указания браузеру, что изображение является картой, в тег  <img> необходимо ввести дополнительный атрибут usemap, определяющий имя карты изображения. Перед этим именем ставится знак "#". Он является ссылкой на описание конфигурации карты, которая задается тегом<map>.
  • Значение атрибута  name тега  <map> должно соответствовать имени в usemap, в отличие от тега <img> мы задаем имя без решетки "#".
  • Размещаем тег <area> , которые определяют интерактивные области в изображении-карте.

Пример записи кода КАРТЫ ИЗОБРАЖЕНИЙ (Image Map) в HTML:


<body>
    <p>
        <img src="image.png" alt="Навигация" usemap="#navigation">
    </p>
    <map name="navigation">
        <area shape = "circle" coords = "200,75,50" href = "green.html" alt = "круг.">
    </map>
</body>

Как же работает тег <area> ?

Каждый тэг <AREA> определяет активные области изображения, которые являются ссылками. Тэг <AREA>задает форму области, ее размеры, устанавливает адрес документа, на который следует сделать ссылку, а также имя окна или фрейма, куда браузер будет загружать документ. Этот тэг всегда располагается в контейнере <map>, который связывает координаты областей с изображением.

Атрибуты тега <area>
  1. Href - Задает адрес документа, на который следует перейти.
  2. Alt - Альтернативный текст для области изображения.
  3. Nohref - Область без ссылки на другой документ. Удобен для вырезания дыр.
  4. Accesskey - Переход к области с помощью комбинации клавиш.
  5. Hreflang - Указывает язык документа, на который ведет ссылка.
  6. Tabindex - Задает последовательность перехода между элементами с помощью клавиши Tab.
  7. Target - Имя окна или фрейма, куда браузер будет загружать документ.
  8. Type- Устанавливает MIME-тип документа, на который ведёт ссылка.
  9. Shape-очертания области. Возможные значения атрибута:
    • rect-прямоугольник;
    • circle-круг;
    • poly-многоугольник.
    • default-указывает всю область.
  10. Coords-Координаты области. Координаты отсчитываются в пикселях от левого верхнего угла изображения, которому соответствует значение 0, 0. Первое число является координатой по горизонтали, второе — по вертикали.


Для прямоугольника (shape="rect") определяется четыре координаты — X1, Y1, X2, Y2.



Для окружности (shape="circle") определяется три координаты — координаты центра окружности (X, Y) и ее радиус (R).



Для полигона (многоугольника) (shape="poly") последовательно указываются координаты каждой вершины (X1, Y1, X2, Y2, и т.д. :)
